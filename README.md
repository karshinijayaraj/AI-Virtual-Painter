# AI-Virtual-Painter
🎨 AI Virtual Painter

An AI-powered Virtual Painter that transforms hand gestures into a digital paintbrush using MediaPipe Hand Tracking and OpenCV. Draw in the air with your index finger, switch colors instantly, clear the canvas, and save your artwork—all without touching a mouse or screen.

🚀 Features
✋ Real-time hand tracking with MediaPipe
🎨 Draw using index finger gestures
🌈 Multiple color palette selection
🧹 Clear canvas with a single gesture/button
💾 Save drawings as image files
⚡ Smooth and responsive drawing experience
📷 Webcam-based interaction

🛠️ Technologies Used
Python
OpenCV
MediaPipe
NumPy

📌 How It Works
Webcam captures live video.
MediaPipe detects and tracks hand landmarks.
The index finger tip is used as a virtual brush.
Color can be selected from the top palette.
Drawings are rendered on a virtual canvas in real time.
Press S to save artwork and Q to quit.
